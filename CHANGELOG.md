# Changelog

All notable changes to **Elara Shop** are documented here.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-04-29

Initial release.

### Added

- Next.js 15 App Router with static export
- React 19 + TypeScript 5.8 setup
- Tailwind CSS 4 with CSS-first `@theme` design tokens
- Dark / light mode with system-preference detection and `localStorage` persistence
- Cart context with add, remove, update quantity, clear, and free-shipping progress bar
- Wishlist context with toggle, persisted state, and live header badge
- Homepage with hero carousel, category showcase, flash offers, brand bar, and product carousels
- Product listing page with filters, sort, and grid/list toggle
- Product detail page with image gallery, tabs (description / ingredients / reviews), related products
- Cart, checkout, order-confirmation, and account pages
- 35+ pre-built pages: blog, careers, press, sustainability, stores, gift cards, referral, affiliates, student program, help, shipping, returns, privacy, terms, cookies, accessibility
- Swiper-powered carousels (hero, products, brands)
- Leaflet-based delivery map widget
- Mega menu (desktop) and slide-out mobile menu with accordion submenus
- SEO metadata with OpenGraph and Twitter card support
- Firebase Hosting configuration

### Documentation

- README.md with quick-start, customization, and deployment guides
- Standalone HTML documentation in `/documentation/index.html`
- Inline TypeScript types throughout
